# Mama Lu's Launch Checklist

Before publishing publicly, verify each item.

## Website
- [ ] Open `index.html` locally.
- [ ] Confirm logo/title look correct.
- [ ] Confirm menu prices are correct.
- [ ] Confirm Carne Picada is $5.75.
- [ ] Confirm Add Beans is $0.25.
- [ ] Confirm taxes are OFF.
- [ ] Confirm beverages are hidden.
- [ ] Confirm salsa is hidden.
- [ ] Confirm call/text buttons are hidden if desired.

## Regular Order Test
- [ ] Add one burrito with beans.
- [ ] Add one burrito without beans.
- [ ] Confirm cart total is correct.
- [ ] Continue to Almost Done.
- [ ] Enter Full Name and phone number.
- [ ] Review order.
- [ ] Send order.
- [ ] Confirm email arrives.

## Office & Group Order Test
- [ ] Add first person.
- [ ] Confirm button changes from Add Person to Add Another Person.
- [ ] Duplicate a person with a new name.
- [ ] Edit a person.
- [ ] Remove a person.
- [ ] Send group order.
- [ ] Confirm email arrives.

## EmailJS
- [ ] Test Mode ON for test orders.
- [ ] Confirm test email says TEST ORDER - DO NOT PREPARE.
- [ ] Set Test Mode OFF before public launch.
- [ ] Send one final real test order.

## Publishing
- [ ] Upload files to GitHub repository.
- [ ] Enable GitHub Pages.
- [ ] Open live URL.
- [ ] Place one live test order from phone.
- [ ] Confirm email arrives.
